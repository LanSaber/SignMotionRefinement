# Archived fitting artifacts

`experiments/` and `visualizations/` are byte-for-byte copies of the GUAVA
fitting outputs from SOKE. Embedded absolute paths are historical provenance;
they were not rewritten. New configs and command defaults point to this
project's artifact tree.

The experiment archive includes the two pretrained parent experiments required
to load the historical frozen meta-implicit pilots. It does not include
unrelated Phoenix, retrieval, trajectory-field, or text-to-sign experiments.
